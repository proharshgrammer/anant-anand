<objective>
Set up Supabase Storage bucket helpers and build a reusable drag-and-drop ImageUploader component used by the Tours and Blog forms. Also create a Destinations admin page (destinations table listing with image upload).
</objective>

<tasks>

<task id="1" type="auto">
**Create Supabase Storage query helpers**

Create `lib/supabase/queries/storage.ts` with:
- `uploadImage(bucket: string, file: File, path: string)` ÔÇö uploads to Supabase Storage, returns public URL
- `deleteImage(bucket: string, path: string)` ÔÇö removes file from bucket
- `listImages(bucket: string, folder: string)` ÔÇö lists files in a folder

Use `createClient()` (browser client) for uploads.
Buckets used: `tour-images`, `blog-images`, `destination-images`, `site-assets`.

Acceptance criteria:
- File exists at `lib/supabase/queries/storage.ts`
- Exports `uploadImage`, `deleteImage`, `listImages`
</task>

<task id="2" type="auto">
**Build ImageUploader component**

Create `components/admin/ImageUploader.tsx` ÔÇö a polished, reusable upload widget:

Props:
```typescript
interface ImageUploaderProps {
  bucket: string;        // Supabase bucket name
  folder?: string;       // sub-folder inside bucket
  value?: string;        // current image URL (for edit forms)
  onChange: (url: string) => void;  // called with public URL on upload
  label?: string;
  accept?: string;       // default: 'image/*'
  maxSizeMB?: number;    // default: 5
}
```

Features:
- Drag-and-drop zone with dashed border, upload icon, and "Drop image here or click to browse" text
- File input (hidden, triggered by click on zone)
- Preview: when `value` is set, show thumbnail (100px) with Ô£ò remove button
- Upload progress: show spinner + "UploadingÔÇª" during upload
- Validation: file size check before upload, show error if too large
- On success: call `onChange(publicUrl)`, show success state briefly
- On error: show red error message
- Alt text input field below the image (required for accessibility): `altText` prop and `onAltChange` handler
- Styled with Tailwind: clean admin aesthetic matching existing shadcn-like style

Acceptance criteria:
- File exists at `components/admin/ImageUploader.tsx`
- Component renders drag-drop zone, preview, alt text field
- `onChange` is called with the public URL after upload
</task>
