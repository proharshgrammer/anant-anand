<objective>
Build the Blog Manager CMS using Tiptap rich-text editor with all required extensions, categories, status management, and SEO fields. Also complete the Enquiries Manager page with status update and notes functionality. Install required Tiptap packages.
</objective>

<tasks>

<task id="1" type="auto">
**Install Tiptap packages and create RichTextEditor component**

Install required Tiptap packages:
```
npm install @tiptap/react @tiptap/starter-kit @tiptap/extension-image @tiptap/extension-link @tiptap/extension-youtube @tiptap/extension-table @tiptap/extension-table-row @tiptap/extension-table-cell @tiptap/extension-table-header @tiptap/extension-character-count @tiptap/extension-placeholder
```

Create `components/admin/RichTextEditor.tsx` — a full-featured 'use client' Tiptap editor wrapper:

Props:
```typescript
interface RichTextEditorProps {
  value: string;         // HTML content
  onChange: (html: string) => void;
  placeholder?: string;
  bucket?: string;       // for image uploads (default: 'blog-images')
}
```

Toolbar (icon buttons grouped by type):
- Group 1: Bold, Italic, Strike
- Group 2: H1, H2, H3
- Group 3: Bullet list, Ordered list, Blockquote, Code block
- Group 4: Link (popover with URL input), Image upload (file picker → upload to Supabase Storage → insert)
- Group 5: YouTube embed (prompt for URL → insert)
- Group 6: Insert Table (3×3 default)
- Group 7: Horizontal rule, Undo, Redo

Editor area:
- Min height 400px, border, rounded, padding
- Character count in footer: "NNN / 50,000 characters"
- Placeholder text when empty

Tiptap extensions to configure:
- StarterKit (default)
- Image (with upload handler that uses `uploadImage()`)
- Link (openOnClick: false, autolink: true)
- Youtube (nocookie: true, width: 640, height: 360)
- Table + TableRow + TableCell + TableHeader (resizable: true)
- CharacterCount (limit: 50000)
- Placeholder

Styles: Add Tiptap content styles in a `<style>` tag (or tailwind prose class) for proper rendering of headings, lists, blockquotes, tables inside the editor.

Acceptance criteria:
- File exists at `components/admin/RichTextEditor.tsx`
- Tiptap packages are in package.json
- Editor renders with all toolbar buttons
- Bold, Italic, H1-H3, Lists work in browser
</task>

<task id="2" type="auto">
**Create reusable SEOFields component**

Create `components/admin/SEOFields.tsx` — a collapsible SEO section for use in both Tour and Blog forms:

Props:
```typescript
interface SEOFieldsProps {
  register: UseFormRegister<any>;
  errors: FieldErrors<any>;
  watch: UseFormWatch<any>;
  setValue: UseFormSetValue<any>;
  prefix?: string;  // field name prefix, default ''
}
```

Fields:
- `meta_title` (text, max 70 chars, char counter below)
- `meta_description` (textarea, max 160 chars, char counter below)
- `focus_keyword` (text, optional)
- `og_image` (ImageUploader, optional)
- `canonical_url` (text, optional)

Collapsible via `<details>/<summary>` or state toggle. Collapsed by default.
Show green/orange/red indicator based on meta_title length (ideal: 30-60 chars).
</task>

<task id="3" type="auto">
**Create Blog Supabase query helpers**

Create `lib/supabase/queries/blog.ts` with:
- `getBlogPosts()` — fetch all posts ordered by created_at DESC: id, title, slug, status, category, published_at, author_name, created_at
- `getBlogPostById(id: string)` — fetch single post with all fields
- `createBlogPost(data: Partial<BlogPost>)` — insert post
- `updateBlogPost(id: string, data: Partial<BlogPost>)` — update post
- `deleteBlogPost(id: string)` — delete post

Use `BlogPost` type from `types/index.ts`.

Also create `lib/supabase/queries/enquiries.ts` with:
- `getEnquiries(filters?: {status?: string})` — fetch enquiries ordered by created_at DESC
- `updateEnquiryStatus(id: string, status: string)` — update status
- `updateEnquiryNotes(id: string, notes: string)` — update internal notes
- `deleteEnquiry(id: string)` — delete enquiry
</task>

<task id="4" type="auto">
**Build BlogForm component**

Create `components/admin/BlogForm.tsx` — using React Hook Form + Zod:

Fields:
- `title` (text, required)
- `slug` (text, auto-generated from title, editable)
- `status` (select: draft | scheduled | published)
- `published_at` (datetime-local input, shown only when status = "scheduled" or "published")
- `category` (text, e.g. "Travel Tips", "Destinations", etc.)
- `tags` (tag input — comma-separated → array)
- `author_name` (text, default: "Anant Anand Team")
- `author_bio` (textarea, optional)
- `excerpt` (textarea, max 300 chars)
- `featured_image` (ImageUploader, bucket: `blog-images`, folder: `featured`)
- `featured_image_alt` (text)
- `content` (RichTextEditor — the Tiptap editor)
- `read_time_minutes` (number, auto-calculated from content length, editable)
- SEO tab: use `<SEOFields>` component
- `schema_type` (select: Article | TravelAgency | BlogPosting)
- `related_tour_ids` — multi-select of tour IDs (for Phase 4; can be placeholder input for now)

Tabs: "Content" | "SEO & Meta"

Props same pattern as TourForm.

Acceptance criteria:
- File exists at `components/admin/BlogForm.tsx`
- All fields render correctly
- RichTextEditor integrated into the form
</task>
