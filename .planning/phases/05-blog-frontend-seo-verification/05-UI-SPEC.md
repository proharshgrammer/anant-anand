## Typography

Declared scale: exactly **4 sizes**, exactly **2 weights**.

| Role | Size | Weight | Line Height | Font Family |
|------|------|--------|-------------|-------------|
| Display (H1) | 32px | 600 (semibold) | 1.2 (leading-tight) | `font-display` (Playfair Display) |
| Section heading (H2) | 20px | 600 (semibold) | 1.3 | `font-display` (Playfair Display) |
| Card title (H3) | 20px | 600 (semibold) | 1.3 | `font-display` (Playfair Display) |
| Subheading (H4) | 20px | 600 (semibold) | 1.3 | `font-display` (Playfair Display) |
| Body | 16px (text-base) | 400 (regular) | 1.625 (leading-relaxed) | `font-body` (Poppins) |
| Small / Meta | 14px (text-sm) | 400 (regular) | 1.5 | `font-body` (Poppins) |
| Badge / Label | 14px (text-sm) | 600 (semibold) | 1.4 | `font-body` (Poppins) |
| Prose content | Inherit prose | Per `@tailwindcss/typography` defaults | 1.75 (prose default) | `font-body` (Poppins) |

Font weights declared: exactly 2 weights — **400 (regular)** for body, small, and meta text; **600 (semibold)** for headings (H1–H4), badges, labels, and all emphasized text. Load only Poppins weights 400 and 600. Heading hierarchy is differentiated by size (32px H1, 20px H2–H4) and font family (Playfair Display for all headings, Poppins for body/text).

### Accent Reserved For (exhaustive list)

Accent color (saffron-500 `#f97316`) is reserved EXCLUSIVELY for:
- **Blog category badges** (non-interactive labels)
- **TOC active heading indicator** (left border highlight + text color)
- **Pagination active page number** (background fill)
- **Blog card date icon** (Clock icon via existing pattern)
- **Navbar active link** (existing pattern from Phase 3)
- **"Book Now" phone CTA button** (existing pattern from Phase 3)
- **WhatsApp floating button** (existing pattern from Phase 3)

Teal-700 (`#0f766e`) is used as a **secondary accent** for:
- Age-group badges (existing pattern)
- Secondary CTAs / "View Details" links (existing pattern)

**Do NOT use saffron for:** Body text, paragraph links (use teal-700 or underline), decorative elements, generic hover states, disabled elements.

### Component Visual Contracts

#### BlogCard
- **Dimensions:** Aspect ratio 16:9 for featured image (h-48 on standard screens)
- **Card wrapper:** `bg-white rounded-2xl overflow-hidden shadow-sm hover:shadow-xl transition-shadow duration-300 border border-gray-100 flex flex-col h-full group`
- **Image:** `w-full h-48 object-cover transform group-hover:scale-105 transition-transform duration-500`
- **Category badge:** Absolute positioned top-left, `bg-white/90 backdrop-blur-sm px-3 py-1 rounded-full text-sm font-semibold text-saffron-800 shadow-sm`
- **Content area:** `p-6 flex-grow flex flex-col`
- **Title:** `text-xl font-semibold text-gray-900 font-display line-clamp-2` with link `hover:text-saffron-600`
- **Excerpt:** `text-gray-600 text-base leading-relaxed line-clamp-3 mt-2 mb-4`
- **Metadata row:** `text-sm text-gray-500 flex items-center` showing date + reading time + author name
- **CTA:** Link to `/blog/[slug]` with "Read More →" text
- **Hover:** Shadow elevation + image scale + link color transition

#### BlogPagination
- **Layout:** Centered flex row with gap-2
- **Page numbers:** `w-10 h-10 rounded-full flex items-center justify-center text-sm font-semibold`
- **Active page:** `bg-saffron-500 text-white`
- **Inactive page:** `bg-white text-gray-600 border border-gray-200 hover:bg-saffron-50 hover:border-saffron-300`
- **Previous/Next:** Text links with arrow icons, disabled state at boundaries (opacity-50 cursor-not-allowed)
- **Ellipsis:** `px-2 text-gray-400` when page count > 7
- **Max visible pages:** 7 (with ellipsis collapse for large sets)

#### BlogTableOfContents
- **Desktop:** Sticky sidebar `sticky top-24 w-64`, positioned to right of content
- **Container:** `bg-white rounded-xl border border-gray-100 shadow-sm p-4`
- **Heading:** "On This Page" in `text-sm font-semibold text-gray-900 uppercase tracking-wider mb-3`
- **TOC items:** Vertical list, `space-y-1`
- **H2 items:** `pl-0 text-sm font-semibold`
- **H3 items:** `pl-4 text-sm text-gray-500`
- **Active state:** `text-saffron-600 font-semibold border-l-2 border-saffron-500 -ml-4 pl-3`
- **Intersection Observer:** Tracks scroll position, sets active heading with 100px root margin
- **Mobile:** Collapsible below content (`lg:hidden`), using details/summary or accordion trigger
- **Smooth scroll:** Click scrolls to heading with 80px offset (sticky nav compensation)

#### Contact Page
- **Layout:** Two-column grid on desktop (`grid grid-cols-1 lg:grid-cols-2 gap-12`)
- **Left column:** Office info (address, phone, email, hours) + EnquiryForm component
- **Right column:** Leaflet map embed (reuse TourMap pattern or `<iframe>` for Google Maps embed from site settings — use static map embed to avoid Leaflet cost for this page)
- **Office info:** Card-style with lucide icons, text-sm
- **Form:** Reuse `components/public/EnquiryForm.tsx` with `sourcePage="/contact"`

#### About / Privacy / Terms Pages
- **Layout:** Centered single column `max-w-3xl mx-auto px-4 sm:px-6 lg:px-8 py-16`
- **Heading:** `text-3xl font-semibold text-gray-900 font-display text-center mb-8`
- **Content:** Typography prose class via `@tailwindcss/typography`
- **Simple navigation:** Back link at bottom if needed
- **Privacy/Terms:** Set `noindex` in metadata (utility pages)

## Interactive Behavior

| Interaction | Element | Behavior |
|-------------|---------|----------|
| TOC navigation | TOC link click | Smooth scroll to heading `#${heading-id}` with 80px offset, update URL hash |
| TOC active tracking | Scroll on blog detail | Intersection Observer highlights current heading, updates active state in TOC sidebar |
| Pagination | Page number click | Client-side `useRouter().push()` to `basePath/page/${n}` (or category variant) |
| Pagination prev/next | Arrow buttons | Disabled on first/last page (opacity-50, cursor-not-allowed) |
| Blog card hover | Image + card overlay | Image scales 105%, shadow elevates from shadow-sm to shadow-xl, CTA link color transitions |
| Category filter | Badge click on listing | Navigate to `/blog/category/[slug]` for filtered view |
| Mobile TOC | Accordion toggle | Collapsible TOC shown below content on viewports < 1024px |
| Image loading | Blog card / detail images | `loading="lazy"` with blur placeholder using `next/image` (or `<img>` with low-quality placeholder bg) |

## Accessibility & SEO Requirements

| Requirement | Implementation |
|-------------|---------------|
| Heading hierarchy | Blog listing: h1 "Our Blog" → h2 per blog card → h3 card title. Blog detail: h1 post title → h2 prose headings → h3 prose subheadings |
| Image alt text | Blog featured_image_alt field in data model — render as `alt={post.featured_image_alt \|\| post.title}` |
| Skip link | Not required (existing site doesn't have one — defer) |
| Focus management | Pagination: focus first card after page change. TOC: focus heading after smooth scroll |
| ARIA labels | Pagination nav: `aria-label="Blog pagination"`. Category filter: `aria-label="Filter by category"` |
| Color contrast | Saffron-500 (#f97316) on white meets WCAG AA for large text (18px+). For small text badges, use saffron-700 (#c2410c) or a darker variant |
| Canonical URLs | All paginated pages: `<link rel="canonical" href="...">` to self. Blog listing page 1: canonical to `/blog` |
| Meta robots | Privacy/Terms pages: `<meta name="robots" content="noindex, follow">` |
| Structured data | Blog detail: `Article` JSON-LD. Blog listing: none (not needed). Contact: `LocalBusiness` JSON-LD |
